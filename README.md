# 🎮 Mario Jump Game

Um mini jogo inspirado no estilo clássico do Mario Jump, desenvolvido utilizando apenas **HTML**, **CSS** e **JavaScript**.

## 🚀 Sobre o projeto

A ideia desse projeto surgiu enquanto eu assistia alguns vídeos sobre mini games no YouTube e pensei em criar minha própria versão, unindo algo que gosto no dia a dia (games 🎮) com meus estudos e evolução como desenvolvedora Front-End.

O projeto ainda está em desenvolvimento e a proposta é continuar melhorando com o tempo, adicionando:
- novas mecânicas
- melhorias visuais
- responsividade
- efeitos sonoros
- otimizações para mobile
- novas funcionalidades
  
## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript 

---

## 📱 Responsividade

O jogo possui uma versão adaptada para dispositivos móveis, atualmente em fase de melhorias.

---

## 🎯 Objetivo

Além da diversão, esse projeto foi criado com o objetivo de praticar:
- manipulação do DOM
- animações com CSS
- lógica de jogo
- colisões
- eventos do JavaScript
- responsividade
- organização de código Front-End
